LAB3-ToDo List Project README
================

프로젝트 소개
----------
- 할 일 입력 칸에 내용을 작성한 후 추가 버튼을 누르면 할 일 목록이 추가되도록 기능 구현
- 추가 버튼을 안 누르고 입력 창에서 엔터만 눌러도 추가되도록 기능 추가
- 등록된 할 일을 수정할 수 있는 기능 구현
- 삭제 버튼으로 추가한 목록 삭제 기능 구현


기술스택
-------------
HTML, CSS, Javascript


역할분담
-----------------
- Create: 조영민
- Read: 박세라
- Update: 신재훈
- Delete: 윤영서
